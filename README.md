# Face Detection Project 👤

## 📌 Project Description
This project is a simple Face Detection system using OpenCV and Python.  
It reads images, processes them, and stores image data and labels for training.

## 📁 Project Structure

facedetection/
│
├── data/
│   ├── images.p
│   ├── labels.p
│
├── images/
│   ├── person1_1.jpg
│   ├── person2_1.jpg
│   └── ...
│
├── your_script.py
├── README.md

## ⚙️ Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Pickle

## ▶️ How to Run

1. Install dependencies:

    pip install opencv-python numpy matplotlib 
   
2. Run the script:

 python your_script.py


## 📊 Output
- Images are converted to grayscale
- Resized to 100x100
- Stored as `images.p` and `labels.p`
